# Argon Documentation

Argon is a Typer-inspired, shell-first CLI framework built on Rich and `prompt_toolkit`.

## Public vs Internal Docs

- Public docs (this folder) define stable v1 behavior and authoring guidance.
- Internal design docs live under `docs/dev/` and are implementation-facing.
