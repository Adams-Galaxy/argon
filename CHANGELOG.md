# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-25
### Added
- Initial public release of Argon with hierarchical command groups, option inference, aliases, hooks, REPL shell, and completion helpers.
- `Shell` and `AsyncShell` interactive experiences plus `run_argv` helper for command-line integration.
- Comprehensive README, usage examples, and basic pytest suite.
